English

Google Research presented TurboQuant at ICLR 2026, an algorithm that addresses the memory bottleneck of Key-Value (KV) caches in large-scale models. TurboQuant combines PolarQuant (random rotation of vector geometry) with the Quantized Johnson-Lindenstrauss (QJL) algorithm, enabling KV cache quantization to just 3 bits with zero accuracy loss. Benchmark testing shows TurboQuant reduces memory usage by 6x and accelerates attention computation by 8x.
