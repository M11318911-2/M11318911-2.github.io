📅 每日行程詳情

D1
8/22

🛫 台北 → 北京 → 烏魯木齊

CA186 13:00-16:15 + CA1295 20:15-00:30+1
搭機前往新疆省會烏魯木齊，專車接機入住五星級酒店。
D2
8/23

🏔️ 烏魯木齊 → 天山天池 → 阿勒泰

車程約2小時至天池，再驅車至阿勒泰
遊覽天山天池（古稱瑤池），融森林、草原、雪山於一體的高山湖泊風光。
天山天池 5A
D3
8/24

🚗 阿勒泰 → 布爾津 → 喀納斯（第一次進）

車程約4-5小時至喀納斯
前往人間淨土喀納斯湖，遊覽臥龍灣、月亮灣、神仙灣。
★ 喀納斯湖 5A（第一次）
D4
8/25

🏘️ 喀納斯 → 禾木（神的自留地）

阿禾公路，車程約2-3小時
前往禾木村，保留圖瓦族古樸民居，白樺林與茅草屋相映，漫步其中彷彿穿越時光。
★ 禾木村 5A
D5
8/26

🌅 禾木（日出）→ 布爾津 → 克拉瑪依

禾木至布爾津約3小時，布爾津至克拉瑪依約3小時
清晨登禾木觀景台賞日出，返回布爾津後前往克拉瑪依。
禾木日出
D6
8/27

🚗 克拉瑪依 → 布爾津 → 五彩灘 → 克拉瑪依

克拉瑪依至布爾津約3小時，布爾津至五彩灘約30分鐘
遊覽額爾齊斯河畔的五彩灘，礫石斑斕如調色板，日光映照下色彩絢麗耀眼。
五彩灘 5A
D7
8/28

🏜️ 克拉瑪依 → 烏爾禾（魔鬼城）→ 克拉瑪依

克拉瑪依至魔鬼城約1.5小時
遊覽烏爾禾魔鬼城的雅丹地貌，風蝕奇岩造型多變，宛如大地上的魔幻世界。
魔鬼城 5A
D8
8/29

🌊 克拉瑪依 → 賽里木湖

車程約5-6小時
長驅前往賽里木湖，碧水蕩漾，雪山倒影湖中，高山湖泊壯麗景色令人心醉。
賽里木湖 5A
D9
8/30

🎨 賽里木湖 → 伊寧（喀贊其）

賽里木湖至伊寧約3小時
遊覽喀贊其乘坐馬車，悠遊草原，感受民族古樸與自然的和諧共生。漫步六星街。
喀贊其（乘馬車）| 六星街
D10
8/31

🦢 伊寧 → 巴音布魯克

車程約4-5小時
前往巴音布魯克草原，天鵝湖與九曲十八彎點綴其中，草原風光壯闊迷人。
巴音布魯克 5A | 天鵝湖 | 九曲十八彎
D11
9/1

🌿 巴音布魯克 → 那拉提草原

車程約3-4小時
前往那拉提草原，體驗騎馬，感受哈薩克族民俗風情與豐富的草原文化。
那拉提草原 5A | 草原騎馬
D12
9/2

☁️ 那拉提（空中草原）→ 獨庫公路北段 → 烏魯木齊

那拉提至獨庫公路北段，再至烏魯木齊（約5-6小時）
遊覽那拉提空中草原觀景台，俯瞰群山與牧場。下午穿越世界最美公路之一——獨庫公路。
空中草原觀景台 | 雪蓮谷 | 獨庫公路
D13
9/3

🏛️ 烏魯木齊（國際大巴扎）→ 北京 → 台北

下午遊覽大巴扎，晚間航班返台
上午遊覽國際大巴扎，具有濃鬱的伊斯蘭建築風格與絲路文化。晚間搭機返台，結束難忘的北疆之旅。
國際大巴扎 | 千回西域（自費）
