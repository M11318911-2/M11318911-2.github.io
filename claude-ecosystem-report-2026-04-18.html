📄 Claude for Word 已全面開放

發布時間：2026 年 4 月 11 日 → 一週內全面開放 Pro/Max

  • 核心功能：Claude 可直接閱讀、編輯 Word 文件，並保留原始排版、字體和標號
  • 局部編輯：使用者可圈選特定段落，指示 Claude 精簡文字、改為被動語態等
  • 追蹤修訂模式：所有修改以 Word 原生「追蹤修訂」呈現，方便逐條接受或拒絕
  • 上下文共享：資訊可與 Claude for Excel、Claude for PowerPoint 互通
  • 可用指令（Slash Commands）：
/check-doc /copy-edit /summarize-contract /flag-issues

主要應用場景

  • 法律合約審查（目前最大用例）
  • 商業文件精簡與潤稿
  • 跨辦公室文件一致性檢查
  • 初稿快速潤色

可用範圍

  • Pro / Max：已全面開放
  • Team / Enterprise：Beta 測試階段
  • 法律 AI 平台（Harvey、Legora）暫無計畫整合

🔒 安全警告：間接提示注入攻擊（Indirect Prompt Injection）

  • 威脅說明：攻擊者可在文件內容、註解或meta資料中隱藏惡意指令，誘使 AI 外洩資料或篡改關鍵條款
  • 攻擊命名：網路安全公司 Oasis Security 將此類攻擊鏈命名為「Claudy Day」攻擊
  • 建議：不要在來源不明或不可信的文件上啟用 Claude 文件存取功能
  • 企業注意：需評估合規要求、文件保留政策、日誌留存方式，並確認是否可限制外部 API 呼叫
